Flask Application Page

Framework Installation

Flask was set up by running the pip install flask command within a Python virtual environment.

Page Development Process

The application is powered by a single Python script named app.py for handling routes, while the HTML content resides in the templates/index.html file, which Flask renders dynamically.

Issues Faced and Resolutions

A key challenge was learning Flask's file structure. This was addressed by adhering to the standard practice of placing HTML files in the templates directory.

Another difficulty was hosting the application. This was resolved by using github.
